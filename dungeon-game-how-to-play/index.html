How to Play Dungeon Game

Objective

Your goal is to navigate through the dungeon, avoid the enemy in blue, and retrieve the Golden Plate.

Game Rules

  • Use the arrow keys to move your character around the dungeon.
  • Avoid the enemy in blue, who will chase you.
  • Find the Sword to defeat the enemy.
  • Collect the Golden Plate to win the game.

Controls

Arrow Keys: Move your character.

Screenshot

How to Download and Run the Game

You can download the Dungeon Game source code from GitHub and run it locally on your machine. Follow these steps:

  1. Visit the GitHub repository and download the code:
    Download the Code from GitHub
  2. Navigate to the downloaded folder.
  3. Compile the Java files: 
    javac *.java
  4. Run the game: 
    java Main

Ensure you have the Java Development Kit (JDK) installed to compile and run the game.
